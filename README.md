# Survivors3D

## 프로젝트 소개
**Survivors3D**는 Unity를 이용해 개발한 3D 생존 액션 게임입니다.  
플레이어는 등장하는 적들을 처치하며 경험치를 획득하고 레벨업을 통해 능력을 강화하며 최대한 오래 생존하는 것을 목표로 합니다. 시간이 지날수록 적의 수와 난이도가 점점 증가하여 긴장감 있는 게임 플레이를 제공합니다.

---

## 주요 기능

### 경험치 및 레벨 시스템
적을 처치하면 경험치가 드롭되며, 플레이어가 이를 획득하면 경험치가 증가합니다. 일정 경험치에 도달하면 레벨업이 이루어지고 플레이어의 능력을 강화할 수 있습니다.

### 이벤트 기반 레벨업 시스템
XPManager에서 **Observer Pattern**을 활용하여 레벨업 이벤트를 발생시키고, UI나 다른 게임 시스템이 이를 구독하여 동작하도록 구현했습니다.

### 난이도 증가 시스템
게임 시간이 지날수록 적의 등장 수와 난이도가 점차 증가하여 플레이어에게 지속적인 도전을 제공합니다.

### 플레이어 이동 및 충돌 처리
플레이어 이동 시 충돌 검사를 통해 장애물과의 상호작용을 처리하여 안정적인 플레이 환경을 구현했습니다.

---

## 사용 기술

- Unity
- C#
- Event 기반 시스템

---

## 프로젝트 목표
이 프로젝트의 목표는 Unity를 활용하여 게임 시스템 설계와 구조화된 코드 작성 방법을 학습하는 것입니다. 또한 이벤트 기반 아키텍처와 기본적인 게임 플레이 시스템을 구현하는 경험을 통해 게임 개발에 대한 이해도를 높이는 것을 목표로 합니다.
